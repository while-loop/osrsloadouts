OSRS Invy
===========

LunchRide Groups service

```bash
$ make deps
```

Installation
------------

#### Go

```bash
$ go get github.com/while-loop/osrsinvy/cmd/...
```

#### From Source

```bash
$ make install
```

Running
-------

#### Docker

```bash
$ docker run --rm while-loop/osrsinvy
```

#### Command line

```bash
$ bin/osrsinvy
```

Testing
-------

```bash
$ make test
```
