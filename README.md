OSRS Loadouts
=============



```bash
$ make deps
```

Installation
------------

#### Go

```bash
$ go get github.com/while-loop/osrsloadouts/cmd/...
```

#### From Source

```bash
$ make install
```

Running
-------

#### Docker

```bash
$ docker run --rm gcr.io/osrs-loadouts/api
```

#### Command line

```bash
$ bin/osrsloadouts
```

Testing
-------

```bash
$ make test
```
